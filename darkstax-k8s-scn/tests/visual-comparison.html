📋 How to Use This Tool

  1. Run the Playwright tests:
    cd darkstax-k8s-scn && npm test
  2. Copy the Figma design screenshot:
    cp ../../screenshot-of-entire-figma-design.jpeg tests/screenshots/figma-design.jpg
  3. Open this file in a browser:
    open tests/visual-comparison.html
  4. Uncomment the image tags in the HTML (lines marked with comments)
  5. Compare side-by-side and identify visual differences
  6. Review test output for specific component gaps

🔍 What to Look For

  • ✅ Layout alignment: Header, panels, canvas positioning
  • ✅ Component sizing: 72px header, 280px menu, 40px buttons
  • ✅ Color accuracy: Grays (#CECECE, #DFDFDF), blue (#00112B)
  • ⚠️ Canvas background: Dotted grid pattern presence
  • ⚠️ Hierarchical lanes: Load, Service, Network, Config labels
  • ⚠️ Connection lines: Dashed lines between resources
  • ⚠️ Resource badges: Count indicators on icons
